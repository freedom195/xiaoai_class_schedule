# 暑假课表系统

为孩子定制暑假课表，通过小爱音箱定时播报任务，识别语音完成回报，配合积分、等级、徽章激励体系，帮助孩子养成良好的学习习惯。

---

## 功能概览

- **课表管理**：日历式排课，支持日视图 / 周视图切换，拖拽调整时间
- **任务类型**：📖学习 / 🏃运动 / 😴休息 / 👁眼保健操 / ✨其他，一键快捷填充
- **重复规则**：仅当天 / 每天 / 每周指定星期，重复任务每天独立计完成
- **时间冲突检测**：保存时自动检测是否与已有课程重叠，有冲突弹确认提示
- **批量清除**：一键清除今天起 / 本周剩余 / 自定义日期范围的课程
- **语音播报**：任务开始前小爱音箱自动播报，结束时提醒
- **语音识别**：孩子说"小爱小爱，我做完了数学"，系统自动记录完成并播报鼓励
- **积分奖励**：完成任务获得积分（可消耗）和经验值（累计升级）
- **等级徽章**：5 个等级称号 + 6 种成就徽章，升级时语音播报
- **多孩子支持**：独立课表、积分、等级，积分榜互相对比
- **兑换中心**：孩子申请奖励，家长在线审批
- **统计报告**：每日完成率折线图、连续天数、积分总览

## 配置小爱音箱

### 第一步：获取设备 ID（mi_did）

**方法一**：打开「米家」App → 小爱音箱设备页 → 右上角「…」→「设备信息」→ 记录「DID」字段

**方法二**：命令行查询

```bash
python -c "
from miservice import MiAccount, MiNAService
import asyncio, aiohttp

async def main():
    async with aiohttp.ClientSession() as s:
        acc = MiAccount(s, '你的手机号', '你的密码', '.mi.token')
        await acc.login('micoapi')
        mina = MiNAService(acc)
        for d in await mina.device_list():
            print(d.get('name'), d.get('deviceID'))

asyncio.run(main())
"
```

### 第二步：在系统中填写

1. 打开系统 → 点击「⚙️ 设置」
2. 填写小米账号、密码、设备 ID
3. 点击「保存并登录」→「测试连接」确认成功
4. 左下角指示灯变绿显示「小爱已连接」

## 使用说明

### 添加孩子

进入「⚙️ 设置」→「孩子管理」→ 填写 emoji 头像和姓名 → 点击「添加」

### 创建课表任务

1. 进入「📅 课表」→ 点击日历空白区域，或点击「+ 新建任务」
2. 选择任务类型（快捷按钮自动填充标题和颜色）
3. 填写任务名称、时间、积分奖励、语音关键词
4. 设置重复规则：
   - **仅当天**：只在所选日期出现
   - **每天**：从创建日起每天出现
   - **每周几**：勾选周几（可多选），每周在这些天出现
5. 点击「保存」；若时段有冲突，弹出确认提示可选择强制保存或重新调整

已创建的任务可直接在日历上**拖拽移动**或**拉伸调整时长**。

### 批量清除课程

1. 点击课表页右上角「🗑 清除课程」
2. 选择快捷范围（今天起 / 明天起 / 本周剩余 / 本月剩余）或手动输入日期
3. 可选择只清除某个孩子的课程
4. 点击「确认删除」

> ⚠️ 重复课程（每天/每周几）的模板会一并删除，所有日期均不再出现。

### 语音播报流程

| 时机 | 播报内容示例 |
|------|-------------|
| 任务开始前 1 分钟 | "小明，现在8点，数学练习开始啦！完成后说'小爱小爱，我做完了数学'就可以记录哦~" |
| 任务结束时 | "数学练习时间到啦，完成了吗？" |
| 孩子回报完成 | "太棒了！小明完成了数学练习，获得20积分！当前积分320分。" |
| 升级时 | "哇！小明升级了！现在是知识探索者，继续加油！" |

### 孩子如何回报完成

对着小爱音箱说以下任意格式均可：

```
小爱小爱，我做完了数学
小爱小爱，数学做完了
小爱小爱，我完成了数学练习
```

系统在 ±30 分钟时间窗口内自动匹配当前任务，每天每项任务只能完成一次。

### 积分与兑换

**孩子申请：** 进入「🎁 兑换」→「申请兑换」→ 填写奖励内容和积分数 → 等待家长审批

**家长审批：** 进入「🎁 兑换」→ 查看待审批列表 → 通过或拒绝（可附留言）

## 等级与徽章

### 等级体系

| 等级 | 称号 | 所需经验 |
|------|------|---------|
| Lv.1 | 暑假新星 | 0 |
| Lv.2 | 学习小达人 | 200 |
| Lv.3 | 知识探索者 | 500 |
| Lv.4 | 暑期冠军 | 1000 |
| Lv.5 | 传奇学霸 | 2000 |

> 完成任务同时获得**积分**（可消耗兑换奖励）和**经验值**（只增不减，用于升级）。

### 成就徽章

| 徽章 | 触发条件 |
|------|---------|
| 🔥 坚持7天 | 连续7天至少完成1项任务 |
| 📚 阅读达人 | 累计完成含"阅读/读书/看书"的任务10次 |
| 🏃 运动健将 | 累计完成含"运动/跑步/游泳/锻炼"的任务10次 |
| 🎵 音乐达人 | 累计完成含"钢琴/音乐/乐器/唱歌"的任务10次 |
| ⭐ 全勤王 | 单日所有计划任务全部完成 |
| 🚀 超额完成 | 单日完成数超过计划数 |

## 常见问题

**Q：端口 8080 已被占用？**

```powershell
Get-NetTCPConnection -LocalPort 8080 | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }
```
然后重新运行 `python main.py`。

**Q：小爱音箱登录失败？**

- 确认账号密码正确（建议用手机号）
- 确认网络可访问小米服务器
- 如开启了两步验证，需先关闭

**Q：说了"做完了"但没识别？**

- 说话时间需在任务时间前后 30 分钟内
- 话语中需包含任务名的关键词（如"数学"）
- 进设置页确认小爱已连接（左下角绿点）

**Q：重复任务删一天能不删其他天吗？**

暂不支持单日跳过，删除重复任务会删除所有日期的实例。可在日历上手动将当天任务时间改到 00:00-00:01 作为跳过标记。

**Q：手机能访问吗？**

可以。启动后终端会显示局域网地址（如 `http://192.168.1.100:8080`），手机连同一 WiFi 后用浏览器打开即可。

## Docker 部署

### 快速启动

```bash
# 构建并启动（后台运行）
docker compose up -d --build

# 查看日志
docker compose logs -f

# 停止
docker compose down
```

启动后访问 **http://localhost:8080**。

### 数据持久化

运行时会自动创建 `./data/` 目录，数据库和小米认证文件均存放在此：

```
data/
├── class_schedule.db   # 数据库
└── .mi.token           # 小米登录 token（登录后自动生成）
```

> 升级镜像时保留 `data/` 目录，数据不会丢失。

### 修改端口

编辑 `docker-compose.yml`，将 `"8080:8080"` 改为 `"你的端口:8080"`：

```yaml
ports:
  - "9090:8080"   # 改为 9090
```

### 设置加密密钥（可选但推荐）

默认用机器特征派生加密密钥。在 Docker 环境中建议显式设置：

```yaml
environment:
  - DATA_DIR=/app/data
  - APP_SECRET=your-strong-random-secret   # 添加此行
```

> 注意：更改 `APP_SECRET` 后已保存的小米密码需要重新填写。

### 手动构建镜像

```bash
# 构建
docker build -t xiaoai-schedule .

# 运行
docker run -d \
  --name xiaoai-schedule \
  -p 8080:8080 \
  -v $(pwd)/data:/app/data \
  -e DATA_DIR=/app/data \
  --restart unless-stopped \
  xiaoai-schedule
```
